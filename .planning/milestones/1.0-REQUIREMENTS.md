## Out of Scope

| Feature | Reason |
|---------|--------|
| 修改文档内容本身 | 纯重构任务，不涉及内容更新 |
| 迁移其他目录内容 | 仅聚焦 6.maps，不包含 2.articles 等 |
| 添加全新内容 | 仅重组已有内容 |
| 知识点拆分为独立文件 | 保持四级标题形式，避免过度碎片化 |
| 视觉设计/主题更改 | 专注结构重组，不涉 UI 修改 |
| 全文搜索功能 | 现有 Nuxt Content 搜索已满足 |
