## 目标

根据 distill-and-archive SKILL.md 第 231 行规范：
> 知识点之间**不使用 `---` 分隔线**，直接连续排列

查找 content/6.maps/ 下所有使用 `---` 作为 #### 四级标题间分隔符的文件，并移除这些分隔线。

## 任务清单

### Task 1: 搜索所有 --- 分隔符
**文件**: 所有 content/6.maps/**/*.md
**动作**: 使用 grep 查找所有 `---` 分隔线，并定位其在 #### 标题间的使用情况
**验证**: 输出匹配的文件和行号
**完成标准**: 获得需要修改的文件列表

### Task 2: 逐个文件清理 --- 分隔符
**文件**: Task 1 中找到的文件
**动作**:
1. 读取文件确认 --- 确实位于 #### 标题之间
2. 移除这些 --- 分隔线
3. 保留其他用途的 ---（如 frontmatter 分隔、章节大分隔）
**验证**: 修改后再次 grep 确认无遗漏
**完成标准**: 所有 #### 间的 --- 被移除

### Task 3: 提交更改
**文件**: 所有修改的文件
**动作**: git add + git commit
**验证**: git status 确认已提交
**完成标准**: 提交完成，状态更新
