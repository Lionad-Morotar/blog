Product Hunt

Product Hunt 产品集

从 Product Hunt 发现的有趣产品工具，按官方分类整理

Productivity

提升工作效率的工具集合

AI notetakers

AI 笔记工具能够录制会议、电话和语音备忘录，然后转录、总结并提取任务，帮助团队、销售人员和学习者跨工具和语言协作。

AI Presentation Software

使用生成式 AI 创建幻灯片的应用程序

Ad blockers

广告拦截器移除广告、跟踪器和干扰，让浏览更清爽、更快速。此类别涵盖浏览器工具、DNS 过滤器和用于隐私、专注和家庭安全的 API。

App switcher

应用切换器集中管理跨应用的启动、切换和操作。支持快捷键、侧边栏和自动化，适合开发者、营销人员和高级用户。

Calendar apps

日历应用帮助你规划时间、追踪事件和同步提醒。此类别汇集了自动排期、集成聊天、支持习惯和 goal planning 的工具。

Compliance software

合规软件整合了保障访问安全、验证身份、管理用户数据和执行策略的工具，帮助团队满足风险、信息安全和人力资源规范。

Content Management Systems

用于创建、组织和发布网站内容的工具。从无头 API 到无代码建站工具和文档引擎，服务于开发者、编辑和团队。

Customer support tools

客户支持软件是企业用于管理和简化客户支持运营的一类软件。通常包括工单管理、在线客服、知识库管理和客户关系管理（CRM）集成等功能。

E-signature apps

电子签名应用支持在文档上进行安全的电子签名。提供签名、分享、追踪和 PDF 处理工具，适用于销售、法律、人力资源和财务部门。

Email clients

邮件客户端管理邮件的发送、接收和组织。此类别涵盖收件箱应用、送达率服务、AI 分类、掩码和开发者工具，确保通信可靠。

File storage and sharing apps

此类应用存储、同步和分享文件，让团队能够安全地编辑、发送和管理内容。涵盖云存储、协作、媒体处理和隐私优先的传输。

Hiring software

招聘软件提供从寻源、筛选、管理候选人到面试、入职直至发薪的全流程工具，适合初创公司、中小企业和全球团队。

Knowledge base software

知识库软件从文档、笔记和应用中存储和提取答案。集中团队知识，加速研究，指导开发者和分析师。

Legal services

法律服务涵盖合同、专利、合规、知识产权和遗产规划等工具，帮助初创公司和中小企业起草、审查和管理法律事务。

Meeting software

让会议更轻松的工具。最好的做法是减少会议，但如果无法避免，那就找一个能帮你规划、总结或提高会议效率的工具。

Note and writing apps

笔记和写作应用在一个地方记录想法、文档和会议。支持协作、语音转文字、思维导图和关联笔记，适用于工作和生活。

PDF Editor

PDF 编辑器是用于修改、编辑和处理 PDF（便携式文档格式）文件的软件应用。用户可以编辑内容，如修改文字、添加或删除图片、重新排列页面以及合并多个 PDF 文件。

Password managers

密码管理器（又称凭证库）安全存储和自动填充登录信息、双因素认证和机密，帮助个人和团队保护账户、分享访问权限并保持合规。

Presentation Software

演示软件包括创建结构化幻灯片的工具——融资路演、销售演示、课程和团队更新。现代平台使用 AI 根据提示或大纲生成幻灯片，帮助团队和初创公司更快制作精美演示文稿。这些是 PowerPoint 风格的演示制作工具，提供模板、布局、品牌主题和实时协作。

Product demo

产品演示工具将屏幕和文字转化为互动导览、视频和指南。团队用它们来解释功能、加速新用户引导和转化潜在客户。

Project management software

项目管理软件帮助团队和个人高效规划、组织和追踪项目。提供集中平台来管理任务、资源、时间表和项目沟通。

  • Shepherd：浏览器生产力扩展，自动追踪浏览习惯并生成每日健康评分，无需账号、100% 免费
  • TypeBoost：AI 写作助手，在任何应用中通过自定义 prompt 处理文本，支持语音输入和 95 种语言翻译

Resume tools

简历工具帮助你制作简历、审核个人资料、评估薪酬和准备面试。适合需要 ATS 友好简历、薪资洞察和练习的求职者。

Scheduling software

排期软件组织预约和时间段，同步日历，自动从消息创建事件，协调团队、工位和广播。

Screenshots and screen recording apps

捕捉屏幕内容，录制操作演示，将点击转化为可分享的可视化内容。适合教程、错误报告、异步反馈和精美的产品演示。

Search

搜索工具跨应用、网站和地图收集和排序信息。帮助快速查找文件、地点和答案，适合研究人员和忙碌的团队。

Security software

安全和隐私工具保护应用、数据和访问。此类别涵盖威胁检测、渗透测试、监控、无密码登录和 PII 保险库，服务团队和用户。

Spreadsheets

电子表格是用于组织、分析和分享数字的数据网格。此类别涵盖规划、报告、应用、计算器和团队协作工具。

Team collaboration software

团队协作软件就像你的数字工作空间，为团队成员提供集中平台来有效沟通、协作和共同工作。这些解决方案提供一系列功能，促进无缝协作并提高团队生产力。

Time tracking apps

时间追踪应用记录工作时长、专注度和工具使用情况，揭示时间去向。适合自由职业者、团队和人力资源部门规划、计费、预算和改善习惯。

Video conferencing

视频会议工具支持虚拟会议、面试、课程和团队同步——录制、转录、翻译、共同创作或构建自定义直播应用，助力远程工作。

Virtual office platforms

虚拟办公平台整合远程团队协作工具：联合办公、会议、排期、工位预订、知识共享和追踪，帮助团队专注和协作。

Web browsers

网页浏览器让你访问和与网站及网络应用互动。此类别涵盖通用浏览器、专注隐私的多账户浏览器、在线编程、自动化和扩展。

Writing assistants

写作助手帮助你起草、翻译、总结和润色文字。将音频或视频转换为清晰的文本，加速研究工作流程并保护隐私。

Engineering & Development

开发者工具与工程基础设施

A/B testing tools

A/B 测试工具运行对照实验（分割测试），比较功能、付费墙、邮件或用户体验。它们使用功能开关、分析和指标来指导产品和增长决策。

AI Code Editors

AI 编程工具和智能 IDE，加速软件开发。这些助手编辑多文件项目、建议代码、自动化 CLI 任务，帮助快速构建移动应用。

AI Code Testing

AI 代码测试工具自动审查、分析和验证代码错误和漏洞。通过在开发早期发现问题来提高代码质量。

AI Coding Agents

AI 编程代理自动提供编程帮助：建议代码、重构、调试，并与编辑器、仓库和 API 集成，加快跨团队和技术栈的构建速度。

AI Databases

AI 数据库存储向量和分析数据，支持快速搜索、聊天和报告。为开发者和团队提供嵌入、实时查询和可扩展的生成式 AI。

Authentication & identity tools

认证和身份工具验证用户、管理登录和保护访问。支持注册、会员、生物识别和应用及网站的安全会话。

Automation tools

自动化工具集中管理任务、减少手动步骤并加速交付。从代码、文档和 CI 到验证码等数据输入障碍，它们提升开发和运维效率。

Browser Automation

无头浏览器和用于自动化浏览器工作流程的其他工具

Cloud Computing Platforms

云计算平台在远程基础设施上托管和扩展应用。提供托管数据库、无服务器运行时、认证、CI/CD 和边缘交付，服务开发者和团队。

Code editors

代码编辑器，有时称为 IDE，是开发者用于编写、编辑和管理软件开发项目源代码的软件工具。这些编辑器提供环境，让开发者高效编写代码、调试错误和轻松浏览代码库。

Code Review Tools

托管代码、自动化审查和运行 CI/CD 的工具。管理拉取请求、测试和部署变更，帮助团队更快交付可靠软件。

Command line tools

命令行工具让你从终端运行代码、构建应用和自动化任务。这些选择适合构建 Web、移动、云和 CI 工作流程的开发者。

Databases and backend frameworks

用于存储数据和支撑服务器逻辑的工具。包括数据库、API 框架和托管平台，用于构建、保护和扩展应用。

Git clients

Git 客户端允许开发者与 Git 交互，Git 是用于追踪代码仓库变更的流行版本控制系统。

Headless CMS software

无头 CMS 工具将内容与展示分离，通过 API 交付到网站、应用和文档。支持多渠道发布、协作和开发者工作流程。

Issue tracking software

问题追踪软件被团队和组织用于管理和追踪各种问题、任务、缺陷或工单的生命周期。提供集中平台来记录、分配和解决问题，确保有效协作和及时解决。

Membership software

会员软件为应用、网站和社区提供门禁访问、支付和用户管理，适合 SaaS、创作者和无代码构建者。

Observability tools

可观测性、事故响应和 AI 驱动自动化的工具。追踪指标、重现问题、保护发布、标记功能并在规模上运行代理。

Predictive AI

预测性 AI 工具分析模式预测结果。此类别整合语言模型、快速推理、搜索和分析，用于研究、交易和营销。

Real-time collaboration infra

为应用启用实时协作的开发者基础设施

Standup bots

站立会议机器人用于促进敏捷或远程团队的每日站会。这些机器人通常与 Slack 或 Microsoft Teams 等消息平台集成，为团队成员分享更新和进度提供结构化格式。

Static site generators

静态网站生成器是用于构建由静态 HTML、CSS 和 JavaScript 文件组成的网站的工具。与从数据库动态生成内容的动态网站不同，静态网站生成器预构建网站页面，然后可以直接提供给用户。

Terminals

终端是运行命令行工具的应用。此类别整合现代模拟器和 AI 副驾驶，用于在 macOS、Windows 和 Linux 上进行编码、服务器运维、SSH 和自动化。

Testing and QA software

测试和质量保证软件端到端验证代码质量：自动化构建、运行测试、监控错误、分类缺陷、测试 API 和模拟浏览器以确保可靠发布。

Unified API

统一 API 工具集中管理认证、数据和 AI 服务，让应用通过单一接口通信。使用共享模式、无服务器后端和跨平台工作流程更快构建。

  • Callio：AI Agent 的 API Gateway，统一接口访问 50+ API（搜索、支付、邮件等），支持 MCP Server 与 Claude Code/Cursor 集成

VPN client

VPN 客户端与远程服务器建立安全和加密连接，允许用户私密浏览互联网、访问受限内容并增强在线安全。

Vibe Coding Tools

Vibe Coding 工具整合 AI 优先编辑器、应用构建器、代理和 Python/Web 技术栈，加速构建、抓取和部署，适合开发者、创作者和团队。

Video hosting platforms

视频托管平台大规模存储、流式传输和管理视频。此类工具处理创作、编辑、交付、分析和分享，服务团队、开发者和营销人员。

Web hosting services

网页托管平台提供基础设施和服务，使你的网站可在互联网上访问。这些平台提供各种托管计划和功能，以适应不同类型的网站和在线项目。

Website analytics

网站分析追踪人们如何找到、使用和在网站及应用上转化。此类工具衡量事件、会话、漏斗和性能，服务团队和隐私需求。

Website builders

网站构建器让用户无需编码或技术专长即可创建网站。这些构建器通常提供用户友好的界面和预制模板，让个人或企业快速轻松地创建和定制网站。

Design & Creative

设计与创意工具

3D & Animation

探索用于创建和动画 3D 场景、动态图形和 VR 的工具。构建模型、绑定、渲染和设计互动视觉效果，用于应用、游戏和网页。

AI Characters

探索能够交谈、教学、辅导和表演的 AI 驱动角色。此类工具模拟人物用于角色扮演、内容、辅导和社区托管。

AI Generative Media

AI 生成媒体工具根据提示创建图像和视频，用智能效果编辑照片，为自拍、产品和品牌提供创意工作流程。

AI Headshot Generators

AI 头像和肖像工具，生成、编辑和换脸，用于个人资料、营销和社交帖子。快速创建逼真的造型、风格和场景。

Avatar generators

AI 头像生成器使用先进算法根据用户输入创建个性化数字头像。这些生成器提供面部特征、发型、服装和配饰的定制选项。

Background removal tools

背景移除工具帮助你轻松移除图像背景。它们使用先进算法将主体与背景分离，让你创建具有透明背景的专业外观图像。

Camera apps

相机应用提供超出设备制造商默认相机应用的额外功能和控制。通常提供手动设置、高级拍摄模式、滤镜、编辑工具和各种创意选项来捕捉和增强你的照片和视频。

Design inspiration websites

从这些策划全球设计师精彩作品的网站获取灵感。然后，可能只是抄袭 Linear。

Design mockups

每个人都知道，设计包装在手机或桌面 UI 中看起来会更酷。查看这些资源，在使用的上下文中展示你的设计。

Design resources

设计师实用资源大礼包。

Digital whiteboards

数字白板是模拟实体白板数字体验的互动软件应用或设备。允许用户使用各种工具和功能在虚拟画布上创建、绘制、书写和协作。

Graphic design tools

平面设计工具汇集用于 UI 模型、照片编辑、动态和互动艺术、网页样式和 AI 视觉的应用，适合设计师、营销人员和创作者。

Icon sets

图标集是设计用于各种数字项目（如网站、移动应用、演示文稿和平面设计项目）的图标集合。

Interface design tools

这些工具对设计师、开发者和产品团队至关重要，可简化网站、移动应用、软件应用和其他数字平台的界面设计、原型制作和迭代流程。

Mobile editing apps

移动编辑应用就像你的便携式照片编辑工作室，让你直接通过移动设备增强和转换照片。这些应用提供一系列强大的编辑工具和功能，展现你图像的最佳效果。

Music Generation

音乐生成工具使用 AI 创作原创曲目或音景。帮助创作者快速制作背景音乐、广告歌或实验性作品。

Photo editing

你拍了一张完美的照片……但它还不够完美。你需要裁剪、滤镜、调整、提亮高光、压低阴影、稍微提高饱和度。试试这些照片编辑应用。

Podcasting Tools

播客工具帮助你创作、托管、配音、编辑和分享有声节目。提供 AI 语音、转录、社交发现、分析和碎片化音频，服务学习者。

Social audio apps

这些应用提供托管或参与基于音频的讨论、播客、访谈和互动会话的平台。

Space design apps

空间设计应用使用 3D、AR 和 AI 帮助规划房间和工作场所。测绘空间、测试布局、挑选家具，支持混合办公和家庭改造。

Stock photo sites

库存照片网站是提供各种专业照片、图形和插图许可的平台。这些网站提供大量高质量图像库，可用于创意项目、网站、博客、广告等。

UI frameworks

UI 框架就像开发者和设计师的设计捷径。它们提供预建组件和样式，让设计和创建美观且响应式的用户界面变得轻松。

User research

用户研究工具集中管理客户反馈、调查和使用洞察，指导产品决策。适合塑造功能、用户体验和路线图的产品经理和设计师。

Video editing

视频编辑涵盖捕捉、组合和增强动态图像的工具——屏幕演示、AI 头像、换脸、3D 场景和自动生成短视频，服务创作者和团队。

  • Cuto：视频剪辑工具
  • Replit Animated Videos：用自然语言描述生成专业动画视频，基于 React 的运动图形，支持 720p/1080p 导出

Voice AI Tools

语音 AI 工具

AI Dictation Apps

AI 语音输入工具让你按一个键，自然说话，即可立即在任何应用中粘贴清晰、结构化的文本。大多数使用按键通话模式（通常是 Fn 键），让你无需切换窗口、打开模态框或复制粘贴即可在任何地方触发语音输入。它们已悄然成为计算机的新输入层：比打字更快、比快捷键更自然，且足够灵活以驱动写作、编码、邮件和 AI 助手的工作流程。

翻译工具自动将口语或书面语言转换为另一种语言。使跨语言交流更快更可靠。
