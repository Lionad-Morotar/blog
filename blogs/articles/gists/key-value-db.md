# 做一个简单的关系数据库

## 起因

在看过《C++ 新经典》后，被其繁杂的特性及语法小小的震撼到了，想进一步深入学习一下（C++ 以及相关知识）。并不是像 Anysort 一样想做出点什么有用的东西，只是一个小小的练习。据《Implement...》博主所说，选择做一个简单的关系型数据库可能是一个做为练习的合适选择，它能提供以下方面的挑战：

* C++
* 面向对象编程
* 经典数据结构及算法
* 内存管理
* 多线程及多进程并发控制
* 网络通讯
* 文件系统
