## 教学设计的认知科学原理

#### 叙事传输（Narrative Transportation）

用具体场景（如 YNAB 连接 Chase Bank）贯穿技术概念，而非抽象定义。
人类大脑对故事的记忆强度是抽象信息的 6-7 倍（双编码理论）。
情境模型让学习者在脑中"预演"使用过程，降低认知负荷。

#### 渐进式术语引入（Progressive Jargon Disclosure）

先让用户熟悉流程（"你登录、选择账户、点击同意"），再引入术语（"这叫 resource owner"）。
**先体验后命名**符合概念形成的心理机制——人类需要先建立直觉，再给直觉贴标签。

#### 认知缺口制造（Information Gap Theory）

"魔法发生了...我会剧透"——故意制造信息缺口，触发好奇心。
当大脑感知到"已知"与"欲知"的差距时，会产生认知性好奇，驱动注意力集中。

#### 情感标记（Emotional Tagging）

"**BAD IDEA!!**"——全大写加双感叹号，视觉和情绪的冲击激活杏仁核，给记忆打上情感标签。
神经科学研究表明，带情感色彩的记忆更容易从长时记忆中提取。

#### 双重编码（Dual Coding）

让用户"先看用户视角，再看开发者视角"——
同一信息通过两种不同的心理表征进行编码，显著增强理解和记忆。

#### 自我解释（Self-Explanation）

不断解释"为什么要这样做"（"为什么不用 URL 传 token"）。
这种因果解释触发**自我生成效应**——人对自己"想明白"的内容记忆更深刻。
