# WebGL

WebGL（Web Graphics Library）是一种在浏览器中进行硬件加速 3D 渲染的 JavaScript API。

## 学习资源

* [WebGL 理论基础](https://webglfundamentals.org/webgl/lessons/zh_cn/)：从基础理论系统学习 WebGL 的完整教程，涵盖基础概念、图像处理、2D/3D 变换、光照、纹理、阴影等主题
* [WebGL2 理论基础](https://webgl2fundamentals.org)：WebGL2 版本的教程，包含更多现代 GPU 特性

## 辅助工具

* [TWGL](https://twgljs.org)：一个轻量级 WebGL 辅助库，简化 WebGL 样板代码
