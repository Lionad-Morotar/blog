## v0.2.51

发布时间：2026-02-24

### 变更内容

- 同步更新至 Claude Code v2.1.51
- 修复在编译后的 Bun 二进制文件中使用 SDK 时出现的 `ReferenceError` 崩溃问题（`bun build --compile`）
- 修复长时间运行的 SDK 会话中由于消息 UUID 跟踪永不清理旧条目导致的内存无限增长问题
- 修复本地斜杠命令输出未返回给 SDK 客户端的问题
- 新增 `task_progress` 事件，用于实时报告后台代理进度，包含累计使用指标、工具计数和持续时间
- 修复 v2 会话 API 中 `session.close()` 在子进程持久化会话数据前终止进程的问题，该问题导致 `resumeSession()` 失效

## v0.2.49

发布时间：2026-02-19

### 变更内容

- 同步更新至 Claude Code v2.1.49
- SDK 模型信息现包含 `supportsEffort`、`supportedEffortLevels` 和 `supportsAdaptiveThinking` 字段，便于消费者发现模型能力
- 当安全检查触发询问响应时，权限建议现在会被填充，使 SDK 消费者能够显示权限选项
- 新增 `ConfigChange` 钩子事件，在会话期间配置文件变更时触发，支持企业安全审计和可选的设置变更拦截

---

## v0.2.47

发布时间：2026-02-18

### 变更内容

- 同步更新至 Claude Code v2.1.47
- 在 `Query` 上新增 `promptSuggestion()` 方法，用于基于当前对话上下文请求提示建议
- 在 `task_notification` 事件中新增 `tool_use_id` 字段，用于将任务完成与原始工具调用关联

---

## v0.2.45

发布时间：2026-02-17

### 变更内容

- 新增对 Claude Sonnet 4.6 的支持
- 在 SDK 流中新增 `task_started` 系统消息，在子代理任务注册时发出
- 修复当后台子代理仍在运行时 `Session.stream()` 过早返回的问题，通过保留中间结果消息直到所有任务完成
- 改进产生大量输出的 shell 命令的内存使用 —— RSS 不再随命令输出大小无限增长
