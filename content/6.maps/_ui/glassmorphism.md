---

#### 玻璃拟态是什么？

一种现代 UI 视觉风格，通过**半透明背景**、**背景模糊**和**细微边框高光**模拟毛玻璃漂浮在空间中的质感，营造层次感和未来感。核心视觉特征包括：

- **半透明填充**：rgba 透明度让背景内容若隐若现
- **背景模糊**：backdrop-filter 创造深度感
- **白色边框高光**：模拟玻璃边缘反光
- **柔和阴影**：增强悬浮感

#### 玻璃拟态的设计起源与流行背景

玻璃拟态在 2020-2021 年随 macOS Big Sur 和 iOS 14 的普及而流行，Apple 将其系统级应用（控制中心、通知中心）作为设计范例。它代表了一种**「深度分层」**的设计哲学——界面元素不是平面的，而是漂浮在 Z 轴空间中的实体。

#### CSS 如何实现玻璃拟态效果？

核心依赖 `backdrop-filter` 属性对元素背后的区域应用滤镜：

```css
.glass {
  /* 半透明背景 */
  background: rgba(255, 255, 255, 0.2);
  /* 背景模糊 - 核心属性 */
  backdrop-filter: blur(10px);
  -webkit-backdrop-filter: blur(10px); /* Safari 兼容 */
  /* 玻璃边缘高光 */
  border: 1px solid rgba(255, 255, 255, 0.3);
  /* 悬浮阴影 */
  box-shadow: 0 4px 30px rgba(0, 0, 0, 0.1);
  border-radius: 16px;
}
```

**注意**：必须设置非完全透明的背景色才能看到模糊效果。

#### 玻璃拟态的适用场景与设计局限

**适用场景**：
- 需要与背景建立层次关系的浮层（卡片、信息面板、模态弹窗）
- 展示「空间感」和「现代感」的界面
- 深色背景下的轻量内容容器

**设计局限**：
- **可读性挑战**：半透明背景会降低文字对比度，对视力不佳用户不友好
- **过度使用风险**：大面积玻璃效果会造成视觉混乱
- **不适用于高优先级交互**：按钮、开关等需要清晰明确的元素不宜使用
- **品牌表达受限**：玻璃效果本身中性，难以传达独特的品牌个性

#### 玻璃拟态与其他设计风格的关系

- **继承自拟物主义**：模拟真实材质（玻璃）的物理特性
- **发展出新拟态（Neumorphism）**：两者都强调「可触知的实体感」
- **被 Apple Vision Pro 的「空间设计」延续**：玻璃的透明、折射、深度感在 visionOS 中得到更极致的表达
