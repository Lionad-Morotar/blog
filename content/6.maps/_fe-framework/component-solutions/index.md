现代前端开发中，组件化是构建可维护、可复用 UI 的核心策略。

## 子领域导航

- [Web Components](web-components) - 原生组件化标准
- [微前端](micro-frontend) - 大型应用的架构方案

## 核心概念

#### Web Components

浏览器原生支持的组件化标准，包含 Custom Elements、Shadow DOM 和 HTML Templates。

#### 微前端

将大型前端应用拆分为独立部署、独立运行的子应用，实现团队自治和技术栈无关。
