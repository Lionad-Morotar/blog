## AI 工具

* [Callio](https://www.callio.dev)：AI Agent 的 API Gateway，统一接口访问 50+ API（搜索、支付、邮件等），支持 MCP Server 与 Claude Code/Cursor 集成

## 生产力工具

* [Shepherd](https://shepherdtime.com/)：浏览器生产力扩展，自动追踪浏览习惯并生成每日健康评分，无需账号、100% 免费
* [TypeBoost](https://www.typeboost.ai/)：AI 写作助手，在任何应用中通过自定义 prompt 处理文本，支持语音输入和 95 种语言翻译

## 视频工具

* [Cuto](https://cuto.video/zh)：视频剪辑工具
* [Replit Animated Videos](https://docs.replit.com/replitai/animated-videos)：用自然语言描述生成专业动画视频，基于 React 的运动图形，支持 720p/1080p 导出
