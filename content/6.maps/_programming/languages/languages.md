#### macOS 预装 Perl 的原因

macOS 底层是 Darwin（BSD Unix），系统内部不少脚本（软件更新、证书管理等）用 Perl 编写。
Perl 在 Python 出现前就是 Unix 文本处理和系统管理的标准工具。
Apple 已在 macOS 12.3 移除 Python 2 和 Ruby，但 Perl 因系统依赖更重而暂时保留。
官方建议开发者不要依赖系统预装脚本语言，应自行安装。
