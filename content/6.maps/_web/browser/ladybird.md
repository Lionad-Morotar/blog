#### Ladybird 是什么？

Ladybird 是一个从零构建的全新浏览器引擎，**非任何现有浏览器的分支**（Not a fork）。

项目由 501(c)(3) 非营利组织支持，致力于解决"整个网络仅由少数几家有自身议程的公司控制的浏览器引擎驱动"的问题。目前网络主要由 Blink（Chrome）、WebKit（Safari）和 Gecko（Firefox）三大引擎主导。

见：[Ladybird 官网](https://ladybird.org/)

#### Ladybird 的技术特点

**完全独立**：不使用 Blink、WebKit、Gecko 等现有引擎的任何代码，基于 Web 标准全新构建。

**语言演进**：原基于 C++，2026 年 2 月宣布采用 Rust 作为 C++ 的继任语言，并使用 AI 代理加速迁移。

**单一专注**：专注于一件事——构建 Web 浏览器，拒绝功能蔓延。

见：[Ladybird 官网](https://ladybird.org/)

#### Ladybird 的运营模式

**无用户变现**：拒绝默认搜索协议、加密代币等模式，完全依赖捐赠和赞助。主要赞助商包括 FUTO、Shopify、Cloudflare 等。

**团队组织**：8 名全职付费工程师 + 大量社区志愿者，保持 18 个月资金储备。

**路线图**：2026 年发布 Linux/macOS Alpha 版（面向开发者和早期用户）；Windows 和移动端非当前优先。

见：[Ladybird 官网](https://ladybird.org/)
