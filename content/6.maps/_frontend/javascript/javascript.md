## 日期时间

#### Temporal API 跨浏览器落地

相比自 1999 年以来饱受诟病的 Date 对象，Temporal 提供全新的日期时间数据结构。Jason
Williams 分享了其诞生背景与巨大工作量。现已通过 polyfill 可用，并正在跨浏览器和服务
端落地。

见：[Web of State of the Browser Day Out](https://remysharp.com/2026/03/18/web-of-state-of-the-browser-day-out)：Remy Sharp 的会议回顾
