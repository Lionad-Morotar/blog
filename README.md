### Issue

若遇到 Fibers 安装失败问题，请按照以下步骤操作：

1. 删除 node_modules 文件夹
2. 删除 package.json 中的 fibers
3. 执行 `npm install`
4. 执行 `npm install --save-dev fibers`
5. 若上一步仍然报错，到 /node_modules/fibers@x.x/ 文件夹中执行 `node ./build.js`
